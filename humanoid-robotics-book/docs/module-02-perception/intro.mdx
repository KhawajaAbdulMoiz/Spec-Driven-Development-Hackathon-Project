# Module 2: Perception and Sensing

In this module, you'll learn how robots perceive and understand their environment through various sensors and perception systems.

## What You'll Learn

- Different types of robot sensors
- Computer vision and image processing
- LiDAR and other 3D sensing technologies
- Audio processing for robots
- Sensor fusion techniques

## Chapters in This Module

- **Chapter 6**: Understanding Sensors - Types and applications
- **Chapter 7**: Computer Vision - How robots see
- **Chapter 8**: LiDAR and 3D Sensing - Depth perception
- **Chapter 9**: Sound and Ultrasonic Sensing - Audio perception
- **Chapter 10**: Sensor Fusion - Combining multiple sensors

## Prerequisites

Complete Module 1: Foundations of Robotics before starting this module.

## Estimated Time

Allow 10-15 hours to complete this module.
