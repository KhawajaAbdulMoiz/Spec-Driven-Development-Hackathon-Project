## Configuration

The `sidebars.ts` file now explicitly lists each module's intro file before the autogenerated chapters:

```typescript
{
  type: 'category',
  label: 'Module 1: Foundations of Robotics',
  collapsible: true,
  collapsed: false,
  items: [
    'module-01-foundations/intro',  // â† Module overview (clickable)
    {type: 'autogenerated', dirName: 'module-01-foundations'}, // â† Chapters
  ],
},
```

3. **First item in dropdown**: Module Overview (intro.mdx) - clickable to view module intro
4. **Following items**: All chapters in order (auto-generated from filesystem)
5. **Clean navigation**: Easy to understand the structure
